# Web Search Skill

Use this skill whenever you need to retrieve up-to-date information from the internet.

## When to Use

- The user asks about events after your knowledge cutoff date
- You need to verify a specific fact with a live source
- The user explicitly asks you to "search" or "look up" something

## Instructions

1. Formulate a precise, concise search query
2. Execute the search using the available search tool
3. Critically evaluate the sources returned
4. Synthesise the information and cite your sources

## Quality Guidelines

- Prefer primary sources (official docs, peer-reviewed papers) over secondary sources
- Always disclose when information comes from a web search
- If results are contradictory, note the discrepancy and present both perspectives
